# End-to-End ETL Pipeline (API → SQL → Dashboard)

## Overview
This project demonstrates a complete ETL pipeline built in Python. 
It extracts data from a public API, transforms and cleans the data, 
and loads it into a SQL database for analysis and reporting.

## Tech Stack
- Python
- Pandas
- Requests (API)
- SQLite / PostgreSQL
- Cron / schedule (automation)

## Pipeline Architecture
Extract → Transform → Load → Report

## Features
- Automated API ingestion
- Data cleaning and validation
- SQL database loading
- Scheduled pipeline runs
- Final report/dashboard

## What This Demonstrates
- ETL pipeline design
- Data engineering fundamentals
- Automation and workflow reliability
- Clean, modular Python code
